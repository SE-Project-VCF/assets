## Issue Summaries

### SCRUM-64 — View Fair Location Details (Nina)
Implemented functionality allowing users to view location details for a selected career fair. The feature displays relevant geographic information and integrates with the location-based fair system so students and employers can easily understand where the event is being held. UI components were added to present the location information clearly, and the view updates dynamically based on the selected fair.

### SCRUM-65 — Set Fair Location (Nina)
Developed the ability for organizers to assign a location to a career fair. This includes input handling for location data and storing the selected location for later retrieval. The feature integrates with the location-based fair workflow and ensures the configured location is available for display in the fair details view.
